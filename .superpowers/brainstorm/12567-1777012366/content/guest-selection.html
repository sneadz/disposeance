Comment ajouter les accompagnants ?

Dans l'étape de sélection des participants, 3 façons d'ajouter un +1

Participants
Nico
+ accompagnant ?
Robin
Sam

A — Champ inline sous le participant

Un input apparaît sous chaque participant sélectionné. Simple, direct. Affiche "Robin + Sa copine" sur la carte.

Participants
Nico
1
Robin +1 Sa copine
2
Sam

B — Bouton +1 / +2 sur la ligne

Un compteur sur chaque participant. Tap sur le badge ouvre un modal pour saisir le prénom. Compact, ne prend pas de place.

Participants
Nico
Robin
Sa copine ✕ + Ajouter
Sam

C — Chips sous le participant

Les accompagnants apparaissent comme des chips supprimables. Un bouton "+ Ajouter" permet d'en ajouter plusieurs. Idéal si parfois plusieurs +1.
